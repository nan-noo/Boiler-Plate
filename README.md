## Boiler - Plate
Login/Register web with node.js(express), react.js.   

before start, create dev.js file in './server/config' and export your mongodb key.  

    module.exports = {mongoURI: 'YOUR_MONGO_URI'}
    
##### Start(dev mode)
    npm run dev
